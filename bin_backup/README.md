# Jojo Player

Jojo Player is a modern open-source media player written in Java, designed to support virtually all video formats through FFmpeg while providing advanced features such as hardware-accelerated playback, AI-powered upscaling,
frame generation, subtitle support, and a customizable user interface.

## Planned Features

- FFmpeg-based playback
- Hardware acceleration (NVDEC, QuickSync, AMF)
- AI Upscaling (Real-ESRGAN)
- AI Frame Generation (RIFE)
- Subtitle Support (SRT, ASS, SSA)
- Playlist Management
- HDR Support
- Picture-in-Picture
- Cross-platform Support
